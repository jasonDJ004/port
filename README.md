# Divine Jason – Personal Portfolio

A modern, responsive personal portfolio website for Divine Jason — Computer Science Graduate, Cloud & Software Enthusiast, and Python & Java Developer.

## 🚀 Features

- **Hero Section** – Name, tagline, contact info, and CTA buttons
- **About Me** – Professional summary
- **Education** – Timeline view of academic background
- **Technical Skills** – Animated progress bars per skill category
- **Projects** – Event Management System & Stock Price Prediction using ML
- **Certifications** – AWS & Java Core certifications
- **Honors & Awards** – Creative Head, CS Dept 2024–2025
- **Strengths** – Key professional strengths
- **Contact Form** – With built-in `localStorage` message database and inbox viewer

## 🗄️ Database

Messages submitted via the Contact Form are stored in the browser's `localStorage` under the key `portfolio_messages`. Click **View Inbox** at the bottom of the Contact section to see all submissions.

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6 (icons)
- Google Fonts – Inter
- localStorage API (client-side database)

## 📂 Project Structure

```
portfolio/
├── index.html        # Main HTML file
├── styles.css        # All styling
├── script.js         # Interactivity + localStorage DB
├── .gitignore

```
